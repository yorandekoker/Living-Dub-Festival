Design a modern festival website UI system for a reggae dub music festival called:

Living Dub Festival 2026

The event is focused on soundsystem culture, reggae, dub, community and connection.

The website should feel like a professional music festival website similar to Dour Festival, Boomtown, or Dub Camp.

Ticket Link Requirement

The festival does not sell tickets directly on the website.

All Buy Tickets / Get Tickets buttons should link to an external ticket platform.

Design requirements:

Buttons link to an external ticket page

The link opens in a new tab

No internal checkout or ticket purchasing system

Treat the ticket button as a call-to-action link
