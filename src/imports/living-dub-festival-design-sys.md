

Design a festival website UI system for a reggae dub music festival called:

Living Dub Festival 2026

The event is focused on soundsystem culture, reggae, dub, community and connection.

The website should feel like a colorful handmade reggae poster turned into a website, inspired by dub soundsystem culture and underground festivals.

Create full desktop layout (1440px) and reusable components.

Avoid corporate or minimal UI styles.

Visual Style

Design the website with a hand-drawn reggae / rasta poster aesthetic inspired by:

Dub soundsystem posters
Street art
Handmade festival flyers
Skate and reggae poster graphics

The interface should feel:

Playful
Creative
Handmade
Warm
Community-driven

Use layered shapes, colorful sections, and playful compositions instead of strict minimal layouts.

Sections can slightly overlap and cards can have slight rotation.

Color Palette; Rasta Inspired: use red yellow green

Primary colors

Reggae Red (#E6392F)
Rasta Yellow (#F7C600)
rasta Green (#138A5A)

reggae festival backgrounds
